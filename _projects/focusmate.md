## 프로젝트 소개

FocusMate는 MediaPipe를 활용해 사용자의 시선, 표정, 자세를 실시간으로 분석하여 집중도를 평가하는 시스템입니다.

## 기술 스택

- **TypeScript** — 타입 안전성이 보장된 메인 로직
- **MediaPipe** — 얼굴 랜드마크 및 포즈 추정 (468 포인트)
- **HTML/CSS** — 반응형 대시보드 UI

## 핵심 기능

1. 실시간 얼굴 랜드마크 추출
2. 시선 방향 벡터 계산
3. 집중도 점수 시계열 시각화
4. 비디오 없이 로컬에서 완전 동작
