## 1. 파이썬 프로그래밍 환경

### 1.1 사용자 인터페이스

사용자 인터페이스(User Interface)는 사용자가 컴퓨터에 명령을 입력하는 환경을 의미한다.

대표적인 방식은 다음과 같다.

| 구분 | 의미 |
|---|---|
| GUI | 그래픽 화면에서 마우스와 아이콘으로 조작하는 방식 |
| CLI | 키보드로 명령어를 입력해 조작하는 방식 |

### 1.2 CLI 환경

CLI(Command Line Interface)는 마우스 클릭이 아니라 **명령어를 직접 입력**해서 컴퓨터를 조작하는 환경이다.

윈도우, macOS, 리눅스 모두 CLI 환경을 기본적으로 제공한다.

예시:

```bash
python input.py
```

위 명령어는 `input.py` 파일을 파이썬으로 실행한다는 뜻이다.

## 2. 화면 입출력

화면 입출력은 사용자가 값을 입력하고, 프로그램이 결과를 출력하는 과정이다.

파이썬에서는 대표적으로 다음 두 함수를 사용한다.

| 함수 | 역할 |
|---|---|
| `input()` | 사용자 입력을 받음 |
| `print()` | 화면에 값을 출력함 |

---

## 3. 표준 입력 함수: `input()`

`input()` 함수는 콘솔 창에서 사용자가 입력한 값을 받아온다.

```python
print("Enter your name:")
somebody = input()
print("Hi", somebody, "How are you today?")
```

실행 예시:

```text
Enter your name:
Sungchul Choi
Hi Sungchul Choi How are you today?
```

`input()`으로 입력받은 값은 기본적으로 **문자열(str)** 로 저장된다.

따라서 숫자로 계산하려면 자료형 변환이 필요하다.

```python
temperature = float(input("온도를 입력하세요: "))
print(temperature)
```

---

## 4. 표준 출력 함수: `print()`

`print()` 함수는 값을 화면에 출력한다.

여러 값을 출력할 때는 콤마(`,`)를 사용할 수 있다.

```python
print("Hello World!", "Hello Again!!!")
```

출력 결과:

```text
Hello World! Hello Again!!!
```

콤마를 사용하면 값 사이에 자동으로 공백이 들어간다.

문자열끼리는 `+` 연산자로도 연결할 수 있다.

```python
print("Hello " + "World!")
```

출력 결과:

```text
Hello World!
```

## 5. Lab: 화씨온도 변환기

섭씨온도를 입력받아 화씨온도로 변환하는 프로그램이다.

변환 공식은 다음과 같다.

```text
화씨온도(°F) = 섭씨온도(°C) × 1.8 + 32
```

예제 코드:

```python
print("본 프로그램은 섭씨온도를 화씨온도로 변환하는 프로그램입니다.")
print("변환하고 싶은 섭씨온도를 입력하세요.")

celsius = input()
fahrenheit = (float(celsius) * 1.8) + 32

print("섭씨온도:", celsius)
print("화씨온도:", fahrenheit)
```

실행 예시:

```text
본 프로그램은 섭씨온도를 화씨온도로 변환하는 프로그램입니다.
변환하고 싶은 섭씨온도를 입력하세요.
32.2
섭씨온도: 32.2
화씨온도: 89.96
```

핵심은 `input()`으로 받은 값이 문자열이기 때문에 계산 전에 `float()`으로 변환해야 한다는 점이다.

## 6. 리스트가 필요한 이유

학생 100명의 성적을 저장한다고 가정하면, 변수를 100개 만드는 것은 비효율적이다.

```python
score1 = 90
score2 = 80
score3 = 70
```

이런 방식은 데이터가 많아질수록 관리하기 어렵다.

그래서 여러 값을 하나의 변수에 저장할 수 있는 **리스트(list)** 를 사용한다.

```python
scores = [90, 80, 70]
```

---

## 7. 리스트의 개념

리스트는 하나의 변수에 여러 값을 저장하는 자료형이다.

```python
colors = ['red', 'blue', 'green']
```

위 코드에서 `colors`는 3개의 값을 가진 리스트이다.

리스트 안에 들어 있는 각각의 값을 **요소(element)** 라고 한다.

---

## 8. 인덱싱

인덱싱(indexing)은 리스트의 특정 요소에 접근하는 방법이다.

파이썬 리스트의 인덱스는 `0`부터 시작한다.

```python
colors = ['red', 'blue', 'green']

print(colors[0])
print(colors[2])
print(len(colors))
```

출력 결과:

```text
red
green
3
```

정리하면 다음과 같다.

| 코드 | 의미 | 결과 |
|---|---|---|
| `colors[0]` | 첫 번째 요소 | `red` |
| `colors[1]` | 두 번째 요소 | `blue` |
| `colors[2]` | 세 번째 요소 | `green` |
| `len(colors)` | 리스트 길이 | `3` |

---

## 9. 슬라이싱

슬라이싱(slicing)은 리스트의 일부를 잘라서 가져오는 기능이다.

기본 문법은 다음과 같다.

```python
리스트명[시작 인덱스:마지막 인덱스]
```

예시:

```python
cities = ['서울', '부산', '인천', '대구', '대전', '광주', '울산', '수원']

print(cities[0:6])
```

출력 결과:

```text
['서울', '부산', '인천', '대구', '대전', '광주']
```

주의할 점은 마지막 인덱스는 포함되지 않는다는 것이다.

즉, `cities[0:6]`은 `0`번부터 `5`번까지 가져온다.

---

## 10. 슬라이싱 활용

### 10.1 시작 또는 끝 생략

```python
cities[0:5]
cities[5:]
cities[:]
```

| 코드 | 의미 |
|---|---|
| `cities[0:5]` | 0번부터 4번까지 |
| `cities[5:]` | 5번부터 끝까지 |
| `cities[:]` | 처음부터 끝까지 전체 복사 |

### 10.2 리버스 인덱스

리버스 인덱스(reverse index)는 뒤에서부터 접근하는 방식이다.

```python
cities[-1]
```

`-1`은 리스트의 마지막 요소를 의미한다.

```python
cities = ['서울', '부산', '인천', '대구', '대전', '광주', '울산', '수원']
print(cities[-1])
```

출력 결과:

```text
수원
```

### 10.3 증가값(step)

슬라이싱에서는 증가값을 지정할 수 있다.

```python
리스트명[시작 인덱스:마지막 인덱스:증가값]
```

예시:

```python
cities[::2]
```

출력 결과:

```text
['서울', '인천', '대전', '울산']
```

역순 출력도 가능하다.

```python
cities[::-1]
```

출력 결과:

```text
['수원', '울산', '광주', '대전', '대구', '인천', '부산', '서울']
```

## 11. 리스트의 연산

### 11.1 덧셈 연산

리스트끼리 `+` 연산을 하면 두 리스트가 합쳐진다.

```python
color1 = ['red', 'blue', 'green']
color2 = ['orange', 'black', 'white']

print(color1 + color2)
```

출력 결과:

```text
['red', 'blue', 'green', 'orange', 'black', 'white']
```

합친 결과를 저장하려면 새로운 변수에 할당해야 한다.

```python
total_color = color1 + color2
```

### 11.2 곱셈 연산

리스트에 숫자를 곱하면 해당 리스트가 반복된다.

```python
color1 = ['red', 'blue', 'green']
print(color1 * 2)
```

출력 결과:

```text
['red', 'blue', 'green', 'red', 'blue', 'green']
```

### 11.3 `in` 연산

`in`은 특정 값이 리스트 안에 있는지 확인하는 연산자이다.

```python
color2 = ['orange', 'black', 'white']

print('blue' in color2)
```

출력 결과:

```text
False
```

## 12. 리스트 추가 및 삭제

리스트에는 값을 추가하거나 삭제하는 여러 함수가 있다.

| 함수 또는 문법 | 기능 | 예시 |
|---|---|---|
| `append()` | 리스트 맨 뒤에 값 추가 | `color.append('white')` |
| `extend()` | 리스트에 다른 리스트를 이어 붙임 | `color.extend(['black', 'purple'])` |
| `insert()` | 특정 위치에 값 추가 | `color.insert(0, 'orange')` |
| `remove()` | 특정 값을 삭제 | `color.remove('red')` |
| `del` | 특정 인덱스 삭제 | `del color[0]` |

### 12.1 `append()`

```python
color = ['red', 'blue', 'green']
color.append('white')

print(color)
```

출력 결과:

```text
['red', 'blue', 'green', 'white']
```

### 12.2 `extend()`

```python
color = ['red', 'blue', 'green']
color.extend(['black', 'purple'])

print(color)
```

출력 결과:

```text
['red', 'blue', 'green', 'black', 'purple']
```

### 12.3 `insert()`

```python
color = ['red', 'blue', 'green']
color.insert(0, 'orange')

print(color)
```

출력 결과:

```text
['orange', 'red', 'blue', 'green']
```

### 12.4 `remove()`

```python
color = ['orange', 'red', 'blue', 'green']
color.remove('red')

print(color)
```

출력 결과:

```text
['orange', 'blue', 'green']
```

### 12.5 인덱스 재할당과 삭제

특정 인덱스의 값을 바꿀 수 있다.

```python
color = ['red', 'blue', 'green']
color[0] = 'orange'

print(color)
```

출력 결과:

```text
['orange', 'blue', 'green']
```

특정 인덱스의 값을 삭제할 때는 `del`을 사용한다.

```python
del color[0]
print(color)
```

출력 결과:

```text
['blue', 'green']
```

## 13. 패킹과 언패킹

### 13.1 패킹

패킹(packing)은 여러 값을 하나의 변수에 묶어 저장하는 것이다.

```python
t = [1, 2, 3]
```

### 13.2 언패킹

언패킹(unpacking)은 리스트 안의 값을 각각의 변수로 꺼내는 것이다.

```python
t = [1, 2, 3]
a, b, c = t

print(t, a, b, c)
```

출력 결과:

```text
[1, 2, 3] 1 2 3
```

언패킹할 때는 리스트 요소 개수와 변수 개수가 같아야 한다.

```python
t = [1, 2, 3]

a, b = t          # 에러
a, b, c, d, e = t # 에러
```

## 14. 이차원 리스트

이차원 리스트는 리스트 안에 리스트가 들어 있는 구조이다.

```python
kor_score = [49, 79, 20, 100, 80]
math_score = [43, 59, 85, 30, 90]
eng_score = [49, 79, 48, 60, 100]

midterm_score = [kor_score, math_score, eng_score]
```

출력하면 다음과 같은 구조가 된다.

```python
[
    [49, 79, 20, 100, 80],
    [43, 59, 85, 30, 90],
    [49, 79, 48, 60, 100]
]
```

이차원 리스트의 값에 접근할 때는 대괄호를 두 번 사용한다.

```python
print(midterm_score[0][2])
```

출력 결과:

```text
20
```

`midterm_score[0][2]`는 0번째 리스트의 2번째 값을 의미한다.

## 15. 리스트의 메모리 관리 방식

파이썬 리스트는 값 자체를 직접 저장한다기보다, 값이 저장된 **메모리 주소**를 저장하는 방식으로 동작한다.

예시:

```python
kor_score = [49, 79, 20, 100, 80]
math_score = [43, 59, 85, 30, 90]
eng_score = [49, 79, 48, 60, 100]

midterm_score = [kor_score, math_score, eng_score]

math_score[0] = 1000

print(midterm_score)
```

출력 결과:

```text
[[49, 79, 20, 100, 80], [1000, 59, 85, 30, 90], [49, 79, 48, 60, 100]]
```

`math_score`의 값을 바꿨는데 `midterm_score` 안의 값도 바뀐다.

이유는 `midterm_score`가 `math_score`의 값을 복사해서 가진 것이 아니라, `math_score`가 가리키는 메모리 주소를 가지고 있기 때문이다.

## 16. `==`와 `is`의 차이

파이썬에서 `==`와 `is`는 의미가 다르다.

| 연산자 | 의미 |
|---|---|
| `==` | 값이 같은지 비교 |
| `is` | 메모리 주소가 같은지 비교 |

예시:

```python
a = 300
b = 300

print(a == b)
print(a is b)
```

결과는 상황에 따라 다를 수 있지만, 핵심은 다음과 같다.

- `a == b`는 값이 같은지 확인한다.
- `a is b`는 같은 객체를 가리키는지 확인한다.

리스트를 다룰 때는 이 차이가 중요하다.

## 17. 리스트 저장 방식의 특징

### 17.1 다양한 자료형 저장 가능

파이썬 리스트에는 서로 다른 자료형을 함께 저장할 수 있다.

```python
a = ["color", 1, 0.2]
```

문자열, 정수, 실수가 하나의 리스트에 함께 들어갈 수 있다.

### 17.2 중첩 리스트 가능

리스트 안에 다른 리스트를 넣을 수도 있다.

```python
a = ["color", 1, 0.2]
color = ['yellow', 'blue', 'green', 'black', 'purple']

a[0] = color

print(a)
```

출력 결과:

```text
[['yellow', 'blue', 'green', 'black', 'purple'], 1, 0.2]
```

이처럼 리스트 안에 리스트가 들어간 구조를 **중첩 리스트**라고 한다.

### 17.3 같은 리스트를 가리키는 경우

```python
a = [5, 4, 3, 2, 1]
b = a

a.sort()

print(b)
```

출력 결과:

```text
[1, 2, 3, 4, 5]
```

`b = a`를 하면 `b`는 `a`의 값을 새로 복사하는 것이 아니라, `a`와 같은 리스트를 가리킨다.

따라서 `a.sort()`로 `a`를 정렬하면 `b`도 정렬된 결과를 보여준다.

새로운 리스트를 할당하면 두 변수는 다른 리스트를 가리키게 된다.

```python
b = [6, 7, 8, 9, 10]
```

## 18. 핵심 정리

| 개념 | 핵심 |
|---|---|
| CLI | 명령어를 직접 입력해 컴퓨터를 조작하는 환경 |
| `input()` | 사용자 입력을 받는 함수 |
| `print()` | 값을 화면에 출력하는 함수 |
| 리스트 | 하나의 변수에 여러 값을 저장하는 자료형 |
| 인덱싱 | 리스트의 특정 요소에 접근 |
| 슬라이싱 | 리스트의 일부를 잘라서 사용 |
| `len()` | 리스트의 길이 반환 |
| `append()` | 리스트 끝에 값 추가 |
| `extend()` | 리스트에 다른 리스트 연결 |
| `insert()` | 특정 위치에 값 추가 |
| `remove()` | 특정 값 삭제 |
| `del` | 특정 인덱스 삭제 |
| 패킹 | 여러 값을 하나의 변수에 저장 |
| 언패킹 | 리스트 값을 여러 변수로 나누어 저장 |
| 이차원 리스트 | 리스트 안에 리스트가 들어간 구조 |
| `==` | 값 비교 |
| `is` | 메모리 주소 비교 |

리스트는 파이썬에서 가장 자주 사용되는 자료형 중 하나이다.  
단순히 여러 값을 저장하는 기능뿐만 아니라, 인덱싱, 슬라이싱, 추가, 삭제, 중첩 리스트 등 다양한 방식으로 활용할 수 있다.  
특히 리스트는 메모리 주소를 기반으로 동작하기 때문에, `b = a`처럼 리스트를 다른 변수에 할당할 때는 두 변수가 같은 리스트를 가리킬 수 있다는 점을 주의해야 한다.
