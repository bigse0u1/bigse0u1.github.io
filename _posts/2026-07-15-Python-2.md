## 1. 화면 입출력

### `input()` 함수

`input()`은 사용자로부터 값을 입력받는 함수이다.  
입력받은 값은 기본적으로 **문자열(str)** 로 저장된다.

```python
name = input()
print(name)
```

숫자로 사용하려면 형 변환이 필요하다.

```python
temperature = float(input("온도를 입력하세요: "))
print(temperature)
```

---

### `print()` 함수

`print()`는 값을 화면에 출력하는 함수이다.  
콤마 `,`를 사용하면 여러 값을 한 번에 출력할 수 있고, 값 사이에 공백이 자동으로 들어간다.

```python
print("Hello", "Python")
```

출력:

```python
Hello Python
```

## 2. 화씨온도 변환기

섭씨온도를 화씨온도로 바꾸는 공식은 다음과 같다.

```python
화씨온도 = (섭씨온도 * 1.8) + 32
```

예시 코드:

```python
celsius = input()
fahrenheit = (float(celsius) * 1.8) + 32

print("섭씨온도:", celsius)
print("화씨온도:", fahrenheit)
```

핵심은 `input()`으로 받은 값은 문자열이므로 계산 전에 `float()`으로 변환해야 한다는 점이다.

## 3. 리스트

리스트는 **하나의 변수에 여러 값을 저장하는 자료형**이다.

```python
colors = ["red", "blue", "green"]
```

학생 100명의 점수를 저장해야 할 때 변수를 100개 만드는 대신, 리스트 하나에 여러 값을 저장할 수 있다.

---

## 4. 인덱싱과 슬라이싱

### 인덱싱

인덱싱은 리스트 안의 특정 값에 접근하는 방법이다.  
파이썬의 인덱스는 `0`부터 시작한다.

```python
colors = ["red", "blue", "green"]

print(colors[0])  # red
print(colors[2])  # green
print(len(colors))  # 3
```

---

### 슬라이싱

슬라이싱은 리스트의 일부를 잘라서 가져오는 방법이다.

```python
cities = ["서울", "부산", "인천", "대구", "대전", "광주", "울산", "수원"]

print(cities[0:3])
```

출력:

```python
['서울', '부산', '인천']
```

슬라이싱은 `시작 인덱스`부터 `끝 인덱스 - 1`까지 가져온다.

```python
print(cities[:])     # 전체 출력
print(cities[::2])   # 2칸 간격으로 출력
print(cities[::-1])  # 역순 출력
```

## 5. 리스트 연산

### 리스트 합치기

```python
color1 = ["red", "blue"]
color2 = ["green", "white"]

print(color1 + color2)
```

출력:

```python
['red', 'blue', 'green', 'white']
```

---

### 리스트 반복

```python
print(color1 * 2)
```

---

### 포함 여부 확인

```python
print("blue" in color1)
```

출력:

```python
True
```

## 6. 리스트 추가와 삭제

| 기능 | 사용법 | 의미 |
|---|---|---|
| 추가 | `append()` | 리스트 맨 뒤에 값 추가 |
| 확장 | `extend()` | 리스트에 다른 리스트를 이어 붙임 |
| 삽입 | `insert()` | 원하는 위치에 값 추가 |
| 삭제 | `remove()` | 특정 값을 삭제 |
| 삭제 | `del` | 특정 인덱스의 값을 삭제 |

```python
color = ["red", "blue", "green"]

color.append("white")
color.insert(0, "orange")
color.remove("red")
del color[0]
```

## 7. 패킹과 언패킹

### 패킹

여러 값을 하나의 변수에 묶는 것이다.

```python
t = [1, 2, 3]
```

### 언패킹

리스트 안의 값을 각각의 변수로 나누는 것이다.

```python
a, b, c = t
print(a, b, c)
```

출력:

```python
1 2 3
```

단, 리스트의 값 개수와 변수 개수가 맞지 않으면 에러가 발생한다.

## 8. 이차원 리스트

이차원 리스트는 리스트 안에 리스트가 들어 있는 구조이다.

```python
kor_score = [49, 79, 20, 100, 80]
math_score = [43, 59, 85, 30, 90]
eng_score = [49, 79, 48, 60, 100]

midterm_score = [kor_score, math_score, eng_score]

print(midterm_score[0][2])
```

출력:

```python
20
```

`midterm_score[0][2]`는 0번째 리스트의 2번째 값을 의미한다.

---

## 9. 리스트의 메모리 관리

파이썬 리스트는 값 자체를 직접 저장하는 것이 아니라, 값이 저장된 **메모리 주소**를 참조한다.

```python
math_score = [43, 59, 85]
midterm_score = [math_score]

math_score[0] = 1000
print(midterm_score)
```

`math_score`를 수정하면 `midterm_score` 안의 값도 함께 바뀔 수 있다.  
이는 두 변수가 같은 리스트 객체를 참조하고 있기 때문이다.
