## 1. 메모리 공간

컴퓨터는 `0`과 `1`, 두 가지 정보만 저장할 수 있으므로 **이진수**를 사용한다.

| 단위 | 의미 |
|---|---|
| 비트(bit) | 이진수 한 자리 |
| 바이트(byte) | 8개의 비트 |
| 킬로바이트(KB) | 1,024바이트 |
| 메가바이트(MB) | 1,024킬로바이트 |

하나의 변수를 메모리에 저장할 때는 해당 변수의 크기만큼 공간을 할당받는다.  
이때 할당받는 메모리 공간의 크기는 변수의 **자료형(data type)** 에 의해 결정된다.

## 2. 변수의 이해

### 2.1 변수와 값

`print()` 함수는 괄호 안의 내용을 화면에 출력하는 함수이다.

```python
a = 7
b = 5

print(a + b)
```

출력 결과:

```결과
12
```

`print(a + b)`는 변수 `a`와 변수 `b`에 저장된 값을 더해서 출력한다.

반면 아래 코드는 다르게 동작한다.

```python
print("a + b")
```

출력 결과:

```결과
a + b
```

`print("a + b")`는 따옴표 안의 문자열을 그대로 출력한다.

즉, 두 코드의 차이는 **따옴표 사용 여부**이다.

| 코드 | 의미 |
|---|---|
| `print(a + b)` | 변수 `a`와 `b`의 값을 더해 출력 |
| `print("a + b")` | 따옴표 안의 문자열을 그대로 출력 |

---

### 2.2 변수와 메모리

**변수(variable)** 는 프로그램에서 특정한 값을 저장하는 공간의 이름이다.

```python
a = 3
```

위 코드는 `a`라는 변수에 `3`이라는 값을 저장한다는 의미이다.

변수를 선언하면 운영체제와 파이썬 인터프리터가 협력하여 메모리 어딘가에 물리적 공간을 할당한다.  
이때 값이 저장되는 위치를 **메모리 주소**라고 한다.

RAM은 **휘발성(volatile)** 성질을 가지고 있기 때문에 컴퓨터 전원이 켜져 있는 동안에만 값이 저장된다.

정리하면 다음과 같다.

- 변수는 값을 저장하는 공간의 이름이다.
- 값은 메모리 어딘가에 저장된다.
- 메모리 주소는 값이 저장된 위치를 의미한다.
- `a = 3`은 `a`라는 변수에 `3`이라는 값을 넣는다는 뜻이다.

---

### 2.3 변수명 선언 규칙

파이썬에서 변수명을 만들 때는 다음 규칙을 지켜야 한다.

| 규칙 | 설명 |
|---|---|
| 사용 가능한 문자 | 알파벳, 숫자, 밑줄 `_` |
| 숫자로 시작 | 불가능 |
| 대소문자 | 구분함(ABC, Abc) |
| 예약어 사용 | 불가능(for, if, else) |
| 권장 방식 | 의미 있는 단어 사용 |

## 3. 기본 자료형

**자료형(data type)** 은 변수에 저장되는 값의 종류를 의미한다.

| 자료형 | 설명 | 예시 |
|---|---|---|
| 정수형 `int` | `0`, `1`, `2`, `-1`, `-2` 등 정수로 한정된 값 | `a = 1` |
| 실수형 `float` | 소수점이 포함된 값 | `a = 1.5` |
| 문자형 `str` | 따옴표 안에 들어간 문자 데이터 | `a = "ABC"` |
| 불린형 `bool` | 참 또는 거짓 | `a = True` |

예시:

```python
a = 1        # int
b = 1.5      # float
c = "ABC"    # str
d = True     # bool
```

## 4. 기본 연산

### 4.1 사칙연산

파이썬에서는 기본적인 사칙연산을 바로 사용할 수 있다.

| 연산 | 연산자 | 예시 | 결과 |
|---|---|---|---|
| 덧셈 | `+` | `25 + 30` | `55` |
| 뺄셈 | `-` | `30 - 12` | `18` |
| 곱셈 | `*` | `50 * 3` | `150` |
| 나눗셈 | `/` | `30 / 5` | `6.0` |

예시:

```python
print(25 + 30)
print(30 - 12)
print(50 * 3)
print(30 / 5)
```

출력 결과:

```결과
55
18
150
6.0
```

---

### 4.2 제곱승

제곱은 `**` 연산자를 사용한다.

```python
print(3 ** 5)
```

출력 결과:

```python
243
```

---

### 4.3 몫과 나머지

몫은 `//`, 나머지는 `%` 연산자를 사용한다.

```python
print(7 // 2)
print(7 % 2)
```

출력 결과:

```결과
3
1
```

| 연산 | 연산자 | 예시 | 결과 |
|---|---|---|---|
| 몫 | `//` | `7 // 2` | `3` |
| 나머지 | `%` | `7 % 2` | `1` |

---

### 4.4 증가/감소 연산

변수의 값을 증가시키거나 감소시킬 때는 `+=`, `-=` 연산자를 사용할 수 있다.

```python
a = 1
a += 1
print(a)
```

출력 결과:

```결과
2
```

`a += 1`은 아래 코드와 같은 의미이다.

```python
a = a + 1
```

감소 연산도 같은 방식으로 사용할 수 있다.

```python
a -= 1
```

`a -= 1`은 아래 코드와 같은 의미이다.

```python
a = a - 1
```

## 5. 자료형 변환

자료형 변환은 값의 자료형을 다른 자료형으로 바꾸는 것이다.

---

### 5.1 정수형과 실수형 변환

`float()` 함수는 정수형을 실수형으로 변환한다.

```python
print(float(10))
```

출력 결과:

```결과
10.0
```

`int()` 함수는 실수형을 정수형으로 변환한다.

```python
print(int(10.7))
```

출력 결과:

```결과
10
```

주의할 점은 `int()`가 반올림을 하는 것이 아니라 소수점 아래를 버린다는 것이다.

---

### 5.2 숫자형과 문자형 변환

`str()` 함수는 숫자형을 문자형으로 변환한다.

```python
a = str(76.3)
print(a)
```

출력 결과:

```결과
76.3
```

반대로 `float()`이나 `int()`를 사용하면 문자형 데이터를 숫자형으로 변환할 수 있다.

```python
a = "76.3"
b = float(a)

print(b)
```

출력 결과:

```결과
76.3
```

문자열끼리 `+` 연산을 하면 숫자 덧셈이 아니라 문자열 연결이 일어난다.

```python
a = str(76.3)
b = str(76.3)

print(a + b)
```

출력 결과:

```결과
76.376.3
```

즉, 문자형 간 `+` 연산은 **문자열 연결(concatenate)** 이다.

또한 숫자형과 문자형을 직접 더하려고 하면 `TypeError`가 발생한다.

```python
a = 10
b = "20"

print(a + b)
```

위 코드는 숫자형과 문자형을 직접 더하려고 했기 때문에 오류가 발생한다.

---

### 5.3 자료형 확인

자료형을 확인할 때는 `type()` 함수를 사용한다.

```python
a = 10
b = 10.5
c = "Python"

print(type(a))
print(type(b))
print(type(c))
```

출력 결과:

```결과
<class 'int'>
<class 'float'>
<class 'str'>
```

| 자료형 | 출력 결과 |
|---|---|
| 정수형 | `<class 'int'>` |
| 실수형 | `<class 'float'>` |
| 문자형 | `<class 'str'>` |
